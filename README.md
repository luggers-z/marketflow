## Build with Maven

Use

```bash
mvn verify site
```

to compile, run tests, measure test coverage and create nifty reports. See `target/site/index.html` or
run `mvn site:run` and open <http://localhost:8080/>.

## Run with Maven

Use

```bash
mvn javafx:run
```

for no-fuzz starting the simulation.
